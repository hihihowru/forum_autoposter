# 虛擬 KOL 儀表板 Wireframes

## 🎨 設計原則

### 整體布局
- **側邊欄導航**: 固定寬度 240px，包含主要功能模組
- **主內容區**: 響應式布局，支援多種螢幕尺寸
- **頂部導航**: 包含用戶信息、設置、通知等
- **實時更新**: 所有數據自動刷新，顯示最後更新時間

### 色彩系統
- **主色調**: #1890ff (Ant Design Blue)
- **成功色**: #52c41a (健康狀態)
- **警告色**: #faad14 (注意事項)
- **錯誤色**: #ff4d4f (異常狀態)
- **中性色**: #f5f5f5 (背景色)

## 🎯 交互設計

### 點擊行為
- **指標卡片**: 點擊顯示詳細數據彈窗
- **圖表**: 支持縮放、篩選時間範圍
- **表格**: 支持排序、篩選、分頁
- **刷新按鈕**: 手動刷新數據
- **設置按鈕**: 打開設置面板
