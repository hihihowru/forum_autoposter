## 🔧 技術棧

### 前端
- React 18 + TypeScript
- Ant Design UI 組件庫
- Vite 構建工具
- React Router 路由管理

### 後端
- FastAPI (Python)
- PostgreSQL 數據庫
- Docker 容器化
- 微服務架構

### 外部服務
- OpenAI GPT API
- CMoney 平台 API
- Serper 新聞搜尋 API
- FinLab 金融數據 API

## 📊 數據流程

1. **觸發器檢測** → 股票數據分析
2. **內容生成** → AI 生成貼文內容
3. **審核流程** → 人工/自動審核
4. **發布管理** → 排程發布到 CMoney
5. **互動追蹤** → 收集互動數據
6. **學習優化** → 分析優化參數
