📋 系統概覽

這是一個基於 AI 的智能內容生成與發布系統，專為金融市場分析和 Key Opinion Leader (KOL) 內容創作而設計。

核心特色

  • 🤖 AI 驅動內容生成：基於 LLM 的個人化內容創作
  • 🎯 多觸發器系統：熱門話題、漲停股等多種觸發機制
  • 👤 深度個人化：KOL 人設管理與風格學習
  • 📊 數據多樣性：避免同批次內容重複
  • 🔄 統一記錄：Google Sheets 格式的完整追蹤
  • 📈 可擴展架構：模組化設計，易於擴展新功能

🔄 工作流程

流程一：熱門話題觸發器

  1. 觸發階段：從 CMoney API 獲取熱門話題列表
  2. KOL 配對階段：根據話題類型匹配適合的 KOL
  3. 相關股票抓取階段：提取話題相關股票代碼
  4. 數據調度階段：根據 KOL 偏好調度數據源
  5. 內容生成階段：生成個人化內容
  6. 記錄階段：記錄到 Google Sheets
  7. 發布階段：發布到 CMoney 平台

流程二：漲停股觸發器

  1. 觸發階段：獲取昨日漲停股列表
  2. 相關股票抓取階段：分析同產業相關股票
  3. KOL 配對階段：根據股票類型匹配 KOL
  4. 數據調度階段：調度股票相關數據
  5. 內容生成階段：生成個人化內容
  6. 記錄階段：記錄到 Google Sheets
  7. 發布階段：發布到平台

🚀 優化方向

短期優化 (1-2個月)

  • 規劃中 內容評分系統
  • 規劃中 A/B 測試機制
  • 規劃中 風險檢測基礎版

中期優化 (3-6個月)

  • 規劃中 動態 KOL 偏好學習
  • 規劃中 智能發布時機預測
  • 規劃中 數據洞察引擎

長期優化 (6個月+)

  • 規劃中 完全自主的內容策略
  • 規劃中 跨平台智能擴展
  • 規劃中 預測性內容生成
